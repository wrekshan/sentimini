Response Type

Response type is used to determining which data to include in graphs.

Open

  • Respond anyway you want
  • These will not be included in graphs.

    • Yes/No

  • Respond with a yes or no
  • These will be included in graphs.

    • 0 to 10

  • Respond with number between 0 and 10
  • These will be included in graphs.


    • All of the responses will be included in tables regardless of the response type.

    Importance

    This used to determine how often to send you a specific text. This is just relative importance.

    In order to determine the text content, Sentimini:
  • Creates a list of possible texts to send
  • Each text is entered the number of times that the importance is set
  • Randomly selects a texts from the list

    • In other words, determining the text content is kind of like a raffle and the importance is how many tickets the text has.

    The higher the number, the more likely you will be sent that text.
