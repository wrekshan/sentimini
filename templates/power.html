Overview

By default, you are signed up to participate in the Open Data Project, which has three goals:
  • Improving user experience
  • Ensuring validity of method (can one actually measure happiness or pain in this way?)
  • Understanding human experience

    • Is this feasible? How many people and datapoints are needed to sensibly test a hypothesis or draw meaningful conclusions from this data?

    To answer this, one would need to do a power analysis. There are two main steps to any power analysis:
  • Stating assumptions about the questions (what type of data? What is the mean or variance of the expected data? What are the drop out rates? What are the statistical tests to be run?)
  • Running calculations to determine how many observations are needed.

    • There are two main ways of running these calculations.
  • There are specific equations for statistical tests to calculate the number of observations needed to achieve a certain probability of observing the expected finding.
  • Running simulations of conducting the analysis using randomized data.

    • Running simulations are prefered for a number of reasons:
  • They establish technical feasibility. To be able to run a simulation is to be able to run the analysis.
  • My mind understands concrete simulations rather than results from equations. In other words, I can just pretend as if I ran the analysis 100 or 1000 times and evaluate how many observations are needed to answer the questions given reasonable assumptions.

    • How

    Ultimately, there are three components to any analysis:
  • Data - Independant Variables and Dependant Variables
  • What you do with the data - The statistical summaries and tests
  • How you put them together

    • Data

    In a quantitative analysis, there are only a few types of data:
  • Dimensional (for example, age, 0-10 scales, height, etc)
  • Categorical (yes or no, gender, favorite color preference, etc)
    A number of things in this world that we care about can be represented in both ways. For example, gender, sleepiness, pain, happines, etc. Quantitative data, especially about the human experience, will never fully capture reality. However, data has to be stored and ideas represented in numbers. How the ideas are represented will deteremine the statistical tests and summaries.
