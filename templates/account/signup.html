Sentimini sends periodic text messages with varying content at semi-random times. You can change the content and the timing at settings.

It displays the responses in downloadable tables and figures. You can view and access these at the home page.

You can learn more at this blog post.

Beta

Sentimini is in beta status. There is a minimum viable product but work still needs to be done to ensure a fulfilling and technological stable experience. As such, it is 'use at your own risk', especially when it comes to data security.

Next major steps:
  • Curated experiences - Sets of words centered around a specific experience (i.e. mindfulness, depression, quitting smoking, dream journal, etc)
  • Greater control of text timing
  • Sharing of data for use with healthcare providers/professional helpers
  • Sharing of data in open source / open science / open data frameworks

    • Using the following use cases at examples:
  • Slow learning / Courses / Guided Experiences
  • Behavior Change Assist
  • Wellness assist - evaluation
  • Welness assist - diagnsosis/communication
  • Learning about this world.
